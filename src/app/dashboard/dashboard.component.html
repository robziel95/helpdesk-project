Description of the application

Application was developed by Robert Zieli≈Ñski in order to gain development skills with Angular

The goal of the application was to create helpdesk system which will allow to manage users and ticket submission. There exist two types of users: administrator and normal user. Administrator has privileges to edit and delete all data of the application (Users and Tickets). Normal users can only edit and delete tickets which they created. They do not have access to tab, which allows to edit and delete users.

Application uses MEAN stack - MongoDB, Express, Angular and NodeJS
